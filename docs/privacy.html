개인정보처리방침

최종 수정일: 2026년 2월 9일

1. 수집하는 개인정보

"내가 두쫀쿠?" 서비스는 다음과 같은 정보를 처리합니다:

  • 사용자가 입력한 이름 (기기 로컬 저장만, 서버 전송 없음)
  • 쿠키 수집 기록 (기기 로컬 저장만)
  • 연속 방문 기록 (기기 로컬 저장만)

2. 개인정보의 이용 목적

수집된 정보는 오직 서비스 기능 제공(쿠키 매칭, 컬렉션 관리, 연속기록 표시)을 위해서만 사용됩니다.

3. 개인정보의 보관 및 파기

모든 데이터는 사용자의 기기에만 로컬 저장되며, 외부 서버로 전송되지 않습니다. 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다.

4. 광고 관련

서비스 내 광고는 토스 앱인토스 광고 시스템(Google AdMob)을 통해 제공됩니다. 광고 관련 데이터 처리는 Google의 개인정보 처리방침을 따릅니다.

5. 문의

개인정보 관련 문의사항이 있으시면 앱 내 고객센터를 이용해 주세요.
