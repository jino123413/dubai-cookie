# 내가 두쫀쿠? — 차별화 비교표

## vs 기존 운세 앱 (tarot-pick, star-fortune, today-fortune, love-fortune)

| 항목 | 내가 두쫀쿠? | tarot-pick | star-fortune |
|------|------------|-----------|-------------|
| 테마 | 두바이 쫀득 쿠키 | 타로 카드 | 별자리 운세 |
| 색상 톤 | 밝은 웜톤 (#FFF8F0) | 다크 (#1A0A14) | 다크 (#0D1B2A) |
| 입력 | 이름 텍스트 입력 | 카드 선택 | 생년월일 |
| 결과 이미지 | 실사풍 쿠키 사진 | 타로 카드 이미지 | 별자리 일러스트 |
| 데이터 구조 | 22종 감정 기반 쿠키 | 22장 메이저 아르카나 | 12 별자리 |
| 핵심 인터랙션 | 쿠키 커팅 애니메이션 | 카드 뒤집기 | 스크롤 페이지 |
| 바이럴 포인트 | 이름 → 친구에게 "너도 해봐" | 카드 공유 | 운세 공유 |

## 경쟁작(푸망/테스트잇) vs 내가 두쫀쿠?

| 항목 | 설명 |
|------|------|
| 입력 간소화 | 10+ 질문 → 이름 1줄 (3초 완료) |
| 결과 다양성 | 16종 고정 → 22종 + 날마다 변동 |
| 이미지 퀄리티 | 카툰 → Flux 실사풍 |
| 리텐션 | 1회성 → 매일 다른 쿠키 + 컬렉션 수집 |
| 플랫폼 | 웹 → 토스 앱 (네이티브 share + 푸시) |
