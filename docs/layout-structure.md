# 내가 두쫀쿠? — 레이아웃 구조

## 3화면 상태머신: home → reveal → result

### Home (이름 입력)
```
┌───────────────────────┐
│    내가 두쫀쿠?        │  ← 헤더
│                       │
│  [연속 N일째 🍪]       │  ← streak chip
│                       │
│    ╭───────╮          │
│    │ 쿠키  │          │  ← 쿠키 아이콘 + 스파클
│    │ 히어로 │          │
│    ╰───────╯          │
│  이름 속에 숨은 감정,   │
│  쫀쿠로 태어나다        │
│                       │
│  22종 중 N종 발견       │
│  [████░░░░░░] 32%     │  ← 컬렉션 진행바
│                       │
│  ┌─────────────────┐  │
│  │ 이름을 입력하세요 │  │  ← 텍스트 인풋
│  └─────────────────┘  │
│                       │
│  [ 내 쫀쿠 확인하기 ]  │  ← CTA
│                       │
│  "오늘도 쫀득하게..."   │  ← 일일 명언
└───────────────────────┘
```

### Reveal (쿠키 커팅 애니메이션)
```
┌───────────────────────┐
│                       │
│     ╭──────╮          │
│     │ 쿠키 │          │  ← 둥근 쿠키
│     │  ↕  │          │  ← 칼 슬라이스 라인
│     ╰──────╯          │
│      ↙    ↘           │  ← 좌우 분리
│                       │
│  "민수"님의 쫀쿠는...   │
│     설렘쫀쿠           │  ← 이름 공개
└───────────────────────┘
```

### Result (단일 스크롤)
```
┌───────────────────────┐
│  ╭────────────╮       │
│  │  쿠키 이미지 │       │  ← 384x576 히어로
│  │  (실사풍)   │       │
│  ╰────────────╯       │
│     ╭ 설렘쫀쿠 ╮       │  ← 이름 뱃지 (이미지에 걸침)
│                       │
│  "당신의 마음엔..."     │  ← 성격 서술 (2~3문장)
│                       │
│  ── 오늘의 한마디 ──   │
│  "두근거림을 멈추지..." │
│                       │
│  [ 친구에게 공유하기 ]  │  ← share() API
│                       │
│  [AD] 다른 이름으로 보기│  ← 광고 버튼
│  광고 시청 후 다른 이름의│
│  쫀쿠를 확인합니다      │
└───────────────────────┘
```

## 차별화 7개 항목

| # | 항목 | 두쫀쿠 | 경쟁작(푸망/테스트잇) |
|---|------|--------|---------------------|
| 1 | 입력 방식 | 이름 한 줄 입력 | 10~16개 MBTI 질문 |
| 2 | 결과 유형 | 22종 + 매일 변동 | 16종 고정 |
| 3 | 이미지 스타일 | Flux 실사풍 쿠키 사진 | 일러스트/카툰 |
| 4 | 재방문 | 매일 다른 쿠키 + 컬렉션 22종 | 1회성 |
| 5 | 내러티브 | 감정→쿠키 데코 (티니핑 방식) | 단순 유형 분류 |
| 6 | 시각 연출 | 쿠키 커팅 clip-path 애니메이션 | 없음 |
| 7 | 색상 톤 | 밝은 따뜻한 쿠키 톤 (#FFF8F0) | 다크/일반 |
